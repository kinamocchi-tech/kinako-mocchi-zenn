## SWE-bench Pro とは何か

「64.3%」という数字を理解するには、まず SWE-bench Pro が何を測るベンチマークなのかを押さえる必要があります。

SWE-bench はもともと 2023 年にプリンストン大学の研究者が作ったベンチマークで、SWE は Software Engineering（ソフトウェアエンジニアリング）の略です。これを Scale AI が発展させ、より実践的で「汚染（データリーク）に強い」Pro 版を 2025 年に公開しました。

SWE-bench Pro が特徴的なのは、**実際の GitHub リポジトリで起きた本物のバグを AI に修正させる** という点です。クイズ的な問題ではなく、現場で実際に発生した不具合が対象になります。問題数は 1,865 問、対象リポジトリは 41 個にのぼります。

車の馬力テストが車の性能を測るように、SWE-bench Pro は AI の「コーディング力」を実務に近い形で測るテストだと考えると分かりやすいでしょう。

## Opus 4.7 を定義する 4つの進化

64.3% という数字は、単なるスケーリング（モデルの大型化）だけで達成されたわけではありません。Opus 4.7 は **4つの直交する軸** で改善されています。

その代表例が「ビジョンの 3 倍解像度化」です。画像の最大サイズが長辺 2,576 ピクセルまで対応し、画面上の細かい文字やボタンまで正確に読み取れるようになりました。コーディングエージェントが UI を見て操作する場面では、この解像度向上が精度に直結します。

複数の改善が独立した軸（直交）で積み上がっているため、特定のタスクに偏らず、幅広いコーディング作業で底上げが効いているのがポイントです。

## ベンチマークだけじゃない — 企業の実測データ

ベンチマークの数字だけを見ると「本当に実務で使えるの？」という疑問が残ります。AI の世界には「ベンチマーク番長」という言葉があり、テストの点数だけ高くて実務では使えないモデルを指します。

そこで重要になるのが、実際に導入した企業の実測データです。

- **Cursor**: 独自の CursorBench で 58% → 70% に上昇（+12 ポイント）
- **楽天**: 社内 SWE-Bench で、本番タスクの解決数が約 3 倍に増加

ベンチマークの数値と実務データの両方で一貫して結果が出ている点が、Opus 4.7 の信頼性を裏づけています。

## 競合比較 — GPT-5.4 との性能と価格

競合と比べたときの立ち位置も整理しておきましょう。

| 項目 | Claude Opus 4.7 | OpenAI GPT-5.4 |
|------|-----------------|----------------|
| SWE-bench Pro | 64.3% | 57.7%（-6.6pt） |
| 入力価格 | $5 / 1M tokens | $2.5 / 1M tokens |
| 出力価格 | $25 / 1M tokens | $15 / 1M tokens |

性能では Opus 4.7 が 6.6 ポイント上回る一方、GPT-5.4 は価格が約半額です。コスト重視の現場では GPT-5.4 が有力な選択肢になります。

注目すべきは、Opus 4.7 が **前世代と同じ価格（入力 $5・出力 $25）を維持したまま性能を上げた** 点です。値上げなしでの性能向上は、導入側にとって素直に嬉しい変化です。なお Anthropic は Opus 系で約 2 ヶ月の更新サイクルを維持しており、競争の激しさがうかがえます。

## 64.3% が意味する「委任の閾値」

この記事で一番伝えたいのは、64.3% という数字の「意味」です。

これは、**人間が常時監視していなくても、難しいコーディング作業を AI に任せられる閾値** を超えた合図だと捉えられます。

その根拠は進化のスピードにあります。約 1 年半前、SWE-bench Pro のトップモデルのスコアは 23% 前後でした。そこから 40 ポイント以上の上昇です。しかもこの進化は単一のベンチマークに閉じておらず、Cursor・楽天・Notion・Devin・XBOW といった分野の異なる企業で一貫して再現されています。

単一テストの数字ではなく、異種環境で再現される結果だからこそ、「AI が同僚として実務を担う時代」が現実になりつつあると言えます。

## まとめ

今回は Claude Opus 4.7 が SWE-bench Pro で記録した 64.3% について深掘りしました。

- SWE-bench Pro は実際の GitHub バグ 1,865 問を解かせる実戦的ベンチマーク
- Opus 4.7 は 4つの直交する進化（ビジョン 3 倍解像度化など）で底上げ
- Cursor +12pt・楽天 3 倍など実務データでも一貫して成果
- GPT-5.4 比で性能は +6.6pt、価格は据え置きで性能向上
- 23% → 64.3% の急進は「AI への委任の閾値」を超えた合図
