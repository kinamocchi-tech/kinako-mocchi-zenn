title: "マーケティング ― 発売48時間とマルチチャネル"
---

審査を通過してスタンプが販売できる状態になったら、いよいよ最後の山場、マーケティングです。本章は「作る」から「売る」へ完全に視点を切り替えます。ここまでの章で身につけた制作スキルは、ここで初めて収益という形になります。

この章は、きなこもっちー（YouTube テックチャンネルのキャラクター。ハムスター = もっちー / セキセイインコ = きなこ）の第1弾スタンプ発売を題材にしていますが、書かれている手順はすべて **あなた自身のスタンプを売るときにそのままコピーできる再現テンプレート** として設計しています。固有名は「事例」として読み替えてください。

:::message
本章の前提となる収益のリアル（第13章）をおさらいします。120 円スタンプの純利益は 100 セット販売で **650〜1,000 円程度**。マーケティングの目的は「1パックで大儲け」ではなく、**第2弾以降の自動通知を届ける購入者リストを積み上げること** です。この前提を握っておくと、本章の施策の優先順位がブレません。
:::

## 大原則：初動48時間がすべてを決める

LINE Creators Market には「新着」ランキングがあり、発売直後の動きが大きい商品ほど上位に表示されます。逆に言えば、**発売から最初の48時間で動かなければ、新着 TOP からこぼれ落ち、その後はほぼ誰の目にも触れなくなります**。

個人クリエイターは 750 万人。その 90% は売上ゼロです。なぜなら大半が「作って申請して終わり」だからです。審査通過後に何の告知もしなければ、LINE のストア内検索だけで発見してもらうのは事実上不可能です。

したがってマーケティングの設計思想はただ一つ。

> **発売日に向けてすべての告知準備を整え、発売の瞬間に全チャネルを同時に撃つ。**

発売は「ゴール」ではなく「打ち上げ花火の点火」です。点火の準備（発売前4週間）と、点火そのもの（発売当日18:00の一斉告知）に最大の労力を割きます。

## 発売タイミング：金曜18:00が最適解

発売日時は、思いつきで決めてはいけません。LINE スタンプは「使ってもらってナンボ」のプロダクトなので、**ユーザーが LINE を最も活発に使う時間帯の直前** に発売するのが鉄則です。

| タイミング | 評価 | 理由 |
|-----------|------|------|
| **金曜 18:00〜19:00** | ◎ 推奨 | 週末の LINE 利用ピークと一致。平日帰宅後にスタンプショップを巡回するユーザーが最多。土日に友人グループで送信される機会が多い |
| 土曜 17:00〜18:00 | ○ 次点 | 週末本番。ただし金曜より新着競合が増える傾向 |
| 火曜・木曜 19:00 | △ 次点 | 平日夜の巡回層は拾えるが、週末への接続が弱い |
| 平日午前・日曜深夜 | ✕ | LINE 利用が落ち込む谷の時間帯。初動が伸びない |

金曜18:00を選ぶ最大の理由は、**「発売 → 週末（土日）の48時間」という最も濃いゴールデンタイムを丸ごと初動期間にできる** からです。土日に友人が「このスタンプ買ったよ」とグループに送れば、それがそのまま新規購入の導線になります。初動48時間の口コミ拡散と、LINE 利用ピークが完全に重なるのです。

:::message alert
発売日時は審査通過後に「販売開始日を予約設定」で指定します。審査が遅延した場合は **発売日も後ろ倒し** にします。「金曜に出したいから審査前に発売日を確定」は厳禁です。審査結果が出てから金曜を予約するのが正しい順序です。
:::

## 発売前4週間タイムライン

初動48時間で爆発させるには、その前の4週間の仕込みがすべてです。各週でやることを明確に分けます。

### Week -4（発売4週前）：基盤を作る

この週は「売る器」を全部用意します。制作と並行して進めます。

- LINE Creators Market アカウント開設・販売者情報登録（銀行口座など振込先も）
- **LINE 公式アカウント開設**（無料枠で月1,000通まで配信可能。これが後で「次作の自動通知先」になる最重要資産）
- 物販を併売するなら SUZURI など商品ページの準備（同日リリース用）
- キャラクターシート確定 → 量産用プロンプト作成（第3〜5章の成果物）

:::message
LINE 公式アカウントは「発売の告知ツール」であると同時に、**事業の心臓部** です。ここに集めた友だちが、第2弾・第3弾の初動を決めます。発売の4週間前という早い段階で必ず開設してください。
:::

### Week -3（発売3週前）：量産と申請

- 第1弾40個を AI で量産・後処理・検証完了（第5〜11章の工程）
- LINE Creators Market へ申請 → 審査開始

審査は **公式目安1〜3営業日ですが、実態は1〜2週間** かかることがあります。Week -3 で申請を済ませ、Week -1 までに審査結果が出ている前提でスケジュールを組みます。

### Week -2（発売2週前）：審査通過と発売予約

- 審査通過 → 販売開始日を翌週金曜18:00に予約設定
- YouTube Shorts 用「制作過程ショート」の撮影・編集
- X / Bluesky でキャラクター画像のチラ見せ投稿を開始（フォロワーの期待値を温める）

審査がまだ通っていなければ、ここで発売日を1週間延期する判断をします。延期は失敗ではなく、織り込み済みのシナリオです。

### Week -1（発売1週前）：曜日別の予告スケジュール

発売週は「毎日少しずつ予告を撃ち、期待を最高潮まで持っていく」期間です。曜日ごとに役割を分けます。

| 曜日 | アクション | 狙い |
|------|----------|------|
| 月 | YouTube Shorts「AIでスタンプ作ってみた #1」公開 | 制作過程で興味を引く |
| 火 | X / Bluesky に「もうすぐ発売」投稿（1枚チラ見せ） | ビジュアル訴求 |
| 水 | YouTube コミュニティタブで予告（「金曜発売です」） | 既存登録者への直接告知 |
| 木 | X / Bluesky にカウントダウン「あと1日」 | 緊急性の演出 |
| 金朝 | TikTok にキャラ紹介ショート投稿 | 当日朝の最後の一押し |

「チラ見せ → カウントダウン → 当日告知」の波を作ることで、発売の瞬間に「待っていた人」が一定数いる状態を作れます。

## 発売当日アクション（金曜）

ここが本章のクライマックスです。**18:00ちょうどに、すべてのチャネルを同時に撃ちます。** 事前に投稿文を全部書いておき、当日は「投稿ボタンを押すだけ」の状態にしておくのが成功の鍵です。

### 18:00 一斉告知（投稿テンプレート）

X / Bluesky 用の告知テンプレートです。これをそのまま自分のスタンプ名・URL に差し替えて使えます。

```text
🎉本日発売🎉
LINEスタンプ「もっちー｜ハムスターの日常スタンプ」
40個 120円で配信開始しました！

おはよう・お疲れさま・大阪弁ツッコミまで
40個入ってどんなシーンでも使える日常スタンプです🐹

ストアはこちら↓
[LINE Creators Market URL]

#LINEスタンプ #新作スタンプ #きなこもっちー
```

:::message
投稿テンプレの設計ポイント
- 1行目に「本日発売」を絵文字付きで置き、視認性を最大化する
- 「40個 120円」と **個数・価格を即明記**（購入判断に直結する情報を隠さない）
- 「どんなシーンでも使える」と用途を一言で伝える
- URL は必ず本文に含める（プロフィールリンク頼みにしない）
- ハッシュタグは3個まで（多すぎると一部 SNS でスパム判定されリーチが落ちる）
:::

### 同時刻実行リスト

18:00に以下を **すべて同時に** 実行します。チェックリスト化して、当日は機械的にこなします。

- [ ] X 投稿（上記テンプレ）
- [ ] Bluesky 投稿（同文面）
- [ ] LINE 公式アカウントから友だちへ告知メッセージ配信
- [ ] YouTube コミュニティタブで投稿
- [ ] YouTube チャンネル概要欄に LINE スタンプ URL を追加
- [ ] X / Bluesky / YouTube のプロフィールリンクをスタンプ URL に差し替え
- [ ] **自分自身が身内 LINE グループにスタンプを実際に送信する**
- [ ] Zenn / note に「LINEスタンプ作りました」記事を公開

:::message alert
**最速の口コミ起爆剤は「身内 LINE グループへの実送信」です。** 友人・家族のグループに自分でスタンプを送るだけで、それを見た数人が「これ何？」と買ってくれます。SNS の不特定多数より、既に関係のある身内のほうが圧倒的に購入率が高い。これは無料で、最も効く施策です。発売直後に必ず実行してください。
:::

### 同日19:30 動画公開

一斉告知から90分後、YouTube に **「AIでキャラクターLINEスタンプ作ってみた」動画（10〜15分）** を公開します。

- 制作過程・キャラデザイン・AI ツール選定・後処理のノウハウを解説
- 動画概要欄に LINE スタンプ購入リンクを設置
- エンドカードでスタンプを告知

なぜ告知（18:00）と動画公開（19:30）をずらすのか。**告知ツイートで「発売した」事実を広めてから、動画で「どう作ったか」という深い文脈を提供する** という二段構えにするためです。動画は SNS 告知より制作に時間がかかる一方、視聴維持・検索流入で長期的に効きます。告知の即効性と動画の持続性、両方を取りに行きます。

## 発売後アクション

打ち上げた後も、火を絶やさないようフォローします。

### Day 1（発売翌・土曜）

- 初日販売数を X で公開（「〇〇人が買ってくれました！」感謝投稿）
- リツイート・引用ポストへの返信を全件対応

数字を公開するのは「社会的証明（みんな買っている）」を作るためです。たとえ少なくても、感謝とともに正直に出すことで応援したくなる空気を醸成します。

### Day 2-3（日・月）

- TikTok / Shorts で「実際に友達に使ってもらった様子」動画を投稿
- X / Bluesky で個別表情のスポット紹介（「この『お疲れさま』が便利」など1枚ずつ）

### Day 4-7（火〜金）

- YouTube コミュニティで「使ってる人スクショ送って！」参加型投稿
- Bluesky テックコミュニティに「AIでスタンプを作った技術話」を投稿
- LINE 公式アカウントから購入者フォローアップメッセージ

### Week 2-4（発売2週後〜1ヶ月）

- 第2弾「きなこスタンプ」制作を **並行開始**（シリーズを止めない）
- KPI 計測（販売数・PV・エンゲージメント）
- 第1弾のレトロスペクティブ（振り返り）実施

## マルチチャネル戦略表

各チャネルには異なる役割があります。「全部に同じ内容を投げる」のではなく、役割に応じて使い分けます。

| チャネル | 役割 | 投稿頻度 |
|---------|------|---------|
| YouTube（長尺） | 「AIでスタンプ作ってみた」動画で深く解説・購入導線 | 発売日 + 1ヶ月後 |
| YouTube Shorts | 制作過程・使用例の短尺動画 | 週2〜3本 |
| YouTube コミュニティ | 既存登録者への直接告知 | 発売前1回 + 当日 + 週1 |
| X (Twitter) | 発売告知・カウントダウン・購入者シェア | 発売前後は毎日 |
| Bluesky | テックコミュニティへの拡散 | 発売後1週間は毎日 |
| TikTok | 若年層への拡散・ショート動画で訴求 | 発売前1回 + 発売後3回 |
| Zenn | 「AIでLINEスタンプを作った」技術記事 | 発売日1本 |
| note | 「個人クリエイターのスタンプ販売記」記事 | 発売1ヶ月後1本 |
| LINE 公式アカウント | 友だち向け告知・次作予告 | 月4通以内 |
| Instagram Reels | Shorts 流用・ビジュアル訴求 | 週1本 |

:::message
個人クリエイターでこれだけのチャネルを回すのは大変に見えますが、**コンテンツは1つ作って使い回します**。制作動画（長尺）→ 切り抜き（Shorts/TikTok/Reels）→ 静止画（X/Bluesky）→ 技術解説（Zenn/note）と、1つの制作プロセスから全チャネル分の素材が生まれます。これが「制作過程をコンテンツ化する」ことの最大の効率性です。
:::

## 差別化の核：YouTube × LINEスタンプの相互集客ループ

ここが、この本の戦略全体で最も重要なポイントです。

LINE Creators Market には個人クリエイターが750万人います。動物キャラだけでもハムスターは1,000種類以上あり、完全な飽和市場です。普通に作って普通に売れば、その他大勢に埋もれます。

では、何で差別化するか。**「テックチャンネルで AI でスタンプを作る過程そのものをコンテンツにする」ことを並走できる人が、750万人のうちほぼゼロ** だという一点です。

| 起点 | 動線 | 効果 |
|------|------|------|
| YouTube 動画 | 概要欄リンク・エンドカード | スタンプ販売数 ↑ |
| YouTube Shorts | 「AIでスタンプ作ってみた」動画 | チャンネル登録数 ↑ + スタンプ告知 |
| LINE スタンプ | 説明欄から YouTube 誘導 | 動画再生数 ↑ |
| LINE 公式アカウント | 購入者をフォロー | 次作の初動数 ↑ |

YouTube が LINE スタンプへ送客し、LINE スタンプが YouTube へ送客する。この **相互集客ループ** が回り始めると、片方が伸びるともう片方も伸びる好循環になります。

あなたが何らかの発信チャネル（技術ブログ・SNS・YouTube）を持っているなら、それは強力な武器です。「作る過程を見せられる」のは、無名の個人クリエイターが大手と戦える数少ない非対称性です。スタンプ単体で勝負せず、**「制作の物語」とセットで売る** ことを徹底してください。

### YouTube チャンネルを持っていない場合（代替導線）

ここまでの相互集客ループは YouTube を軸に説明してきましたが、**YouTube チャンネルがなくても「制作の物語」戦略はそのまま成立します**。重要なのは「YouTube かどうか」ではなく「**作る過程を見せられる発信面を 1 つ持つこと**」です。あなたの手持ちのチャネルに置き換えてください。

| 手持ちチャネル | 「制作の物語」の見せ方 | LINE への送客動線 |
|--------------|--------------------|------------------|
| **X（旧 Twitter）** | 制作過程をスレッドで連投（プロンプト失敗 → 改善 → 完成の Before/After 画像が伸びやすい）。`#AIイラスト` `#LINEスタンプ` 等で発見性を確保 | プロフィール固定ポスト + 各スレッド末尾に販売 URL |
| **TikTok / Instagram Reels** | 「AI でキャラ作ってみた」「透過の罠」を 15〜30 秒の縦型ショートに。YouTube Shorts と同じ素材を流用 | プロフィールリンク（リンク集ツール）→ LINE スタンプ |
| **技術ブログ（Zenn / Qiita / note）** | 本書のような「制作手順・失敗と対策」を記事化。検索流入が長く効く（ストック型） | 記事末尾に販売 URL + 作者プロフィール |
| **Pinterest / Threads** | 完成スタンプ・キャラ立ち絵をビジュアル投稿。海外（台湾・タイ）にも届く | ピン/投稿リンクから LINE STORE |

ポイントは 3 つです。

1. **1 つの発信面に集中する**：複数を薄く回すより、自分が続けやすい 1 チャネルで「制作の物語」を出し続けるほうが、相互集客ループは回り始めます（YouTube である必要はない）。
2. **ストック型（ブログ）とフロー型（SNS）を 1 つずつ**：理想は「検索で長く効くブログ 1 本 + 拡散が速い SNS 1 本」。本書を読んでいるあなたは、すでに技術記事を書ける素地があるはずです。
3. **送客動線は必ず固定する**：どのチャネルでも「プロフィール or 固定投稿に販売 URL」を置き、毎回の投稿末尾にも URL を添えます。これは YouTube の概要欄リンクと同じ役割です。

> YouTube を持っている人はそれが最強の武器ですが、**持っていなくても「作る過程をコンテンツ化する」非対称性は誰でも作れます**。チャネルは手段、本質は「制作の物語とセットで売る」ことです。

## KPI 計測：何を見るか、なぜ見るか

施策は「やりっぱなし」にせず、必ず数字で振り返ります。発売30日後に以下を計測します。

| 指標 | 30日目標 | 計測ツール |
|------|----------|-----------|
| 販売数（セット） | 100 | LINE Creators Market ダッシュボード |
| LCM ページ PV | 1,000 | LCM アナリティクス |
| X 投稿インプレッション | 10,000 累計 | X Analytics |
| YouTube 概要欄リンク CTR | 50 クリック | YouTube Studio |
| LINE 公式アカウント友だち数 | 50 | LINE 公式アカウントマネージャー |
| Bluesky エンゲージメント | 50 累計 | Bluesky 投稿分析 |

ここで最も注目すべきは、**販売数100セットそのものではありません**。第13章で見たとおり、100セットの純利益は650〜1,000円。これ単体では「儲かった」とは言えません。

本当に追うべきは **「LINE 公式アカウント友だち数」と「次作の通知を届けられる購入者数」** です。

> **目的は「1パックを何本売るか」ではなく、「次作の通知先を何人増やせるか」。**

100セットの販売は、利益のためというより「次作の自動通知配信先を増やすための投資」と位置づけます。この発想の転換ができると、目先の販売数に一喜一憂せず、長期の資産形成として淡々と続けられます。

## 第2弾以降の販促ループ：通知が積み上がる構造

なぜ「通知先を増やす」ことがそれほど重要なのか。それは、**LINE スタンプ販売がシリーズを重ねるほど有利になる構造** を持っているからです。

```text
第1弾発売 → 30日 KPI 計測 → レトロスペクティブ
   ↓
第2弾「きなこ」制作（第1弾の購入者へ自動通知が効く）
   ↓
第2弾発売 → 第1弾より高い初動が期待できる
   ↓
第3弾「アニメコラボ」制作 → さらに通知先が積み上がる
```

LINE は、あるクリエイターのスタンプを購入したユーザーに対し、同じクリエイターの新作が出たときに **自動で通知** します。つまり、第1弾を100人が買えば、第2弾の発売時にその100人へ自動で告知が届きます。第2弾を更に150人が買えば、第3弾には合算した分が届きます。

**第1弾 → 第2弾 → 第3弾と続けるたびに、ゼロから集客し直す必要がなくなり、初動が積み上がっていく** のです。これがシリーズ化の最大の収益エンジンです。月収1〜3万円ラインに到達するのは、たいてい第3弾以降。累積効果が効いてくる地点です。

:::message alert
逆に言えば、**シリーズの放置（3〜6ヶ月以上の沈黙）は、積み上げた通知資産の劣化に直結** します。せっかく集めた購入者リストも、新作が出続けなければ忘れられます。第1弾を出したら、間を空けず第2弾の制作に入る。これがシリーズ運用の鉄則です。
:::

## リスク対応：初動が伸びなかったら

すべてが計画どおりに行くとは限りません。発売直後の動きに応じて、対応を用意しておきます。

| シナリオ | 対応 |
|---------|------|
| 初日販売数 < 10 | X / Bluesky で再告知・身内へのお願い再送・YouTube コミュニティで再投稿 |
| 初動好調（> 30） | 「ありがとう投稿」を即アップ・全 SNS への露出を強化して波に乗る |
| 審査リジェクト | 修正対応 → 発売日を1週間延期・告知も延期 |
| 海外売上が伸びる | 英語版・タイ語版テキストスタンプを第3弾以降で検討 |

初日が伸びなくても落ち込む必要はありません。最初の1パックは「販促の練習台」でもあります。ここで得た KPI とレトロスペクティブが、第2弾の精度を上げます。**完璧な1発より、改善し続ける連発** が個人クリエイターの勝ち筋です。

## この章のまとめ

- **初動48時間が新着 TOP 表示の鍵**。発売日に向けてすべての告知準備を整え、発売の瞬間に全チャネルを同時に撃つのが最優先。
- **金曜18:00発売** が最適。週末の LINE 利用ピークと、初動48時間（土日）のゴールデンタイムが完全に重なる。
- 発売前4週間で「基盤づくり（Week -4）→ 量産・申請（Week -3）→ 審査通過・発売予約（Week -2）→ 曜日別予告（Week -1）」と段階的に仕込む。
- 当日18:00に X・Bluesky・LINE 公式・YouTube・記事を **同時告知**。最速の口コミ起爆剤は **身内 LINE グループへの実送信**。19:30 に制作動画を公開。
- 差別化の核は **YouTube × LINE スタンプの相互集客ループ**。「制作の物語」とセットで売ることが、無名の個人が飽和市場で戦える非対称性。
- 追うべき KPI は販売数より **「次作の通知先（友だち・購入者）の増加」**。シリーズを重ねるほど自動通知が積み上がり、初動が有利になる。

ここまでで、企画から制作、申請、収益化、そしてマーケティングまで、AI で LINE スタンプを作って売る一連のパイプラインを一周しました。次章（最終章）では、本書全体を振り返りつつ、この手作業の連続をどう **自動化・仕組み化** して、継続的なスタンプ事業へと発展させていくかを展望します。
