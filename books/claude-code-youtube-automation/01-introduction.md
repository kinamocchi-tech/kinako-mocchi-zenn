## なぜ「AI駆動のYouTubeチャンネル制作」なのか

YouTubeチャンネルを運営するには、テーマ選定・リサーチ・台本作成・音声収録・画像制作・動画編集・メタデータ設定・公開スケジュール管理など、膨大な作業が必要です。週2本の更新ペースを維持しながら品質を保つことは、個人クリエイターにとって最大の課題です。

Claude Code（Anthropic が提供する CLI ベースの AI コーディングエージェント）は、Agent Skills（再利用可能なワークフロー定義。Ch.03 で詳述）、Scheduled Tasks（定期自動実行。Ch.15 で詳述）、Agent Teams（並列サブエージェント。Ch.03 で詳述）といった機能を組み合わせることで、 **複雑なワークフロー全体を自律的に実行** できます。本書では、この能力を最大限に活用して制作パイプライン全体をほぼ自動化するシステムを構築します。

:::message
**設計思想: 「完全自動で80点」より「半自動で95点」**

AI が得意な作業（テキスト処理・音声合成・画像生成・定型 API 操作）は徹底的に自動化しつつ、人間の判断が不可欠な部分（ファクトチェック・品質確認・公開判定）には必ず人間が介在します。
:::

## 実例: 「きなこもっちーのテック深掘り」チャンネル

本書は、実際に稼働している YouTube チャンネル **「きなこもっちーのテック深掘り」** の制作パイプラインをベースにしています。

| 項目 | 内容 |
|------|------|
| ジャンル | AI/LLM を中心としたテック解説 |
| 形式 | 2体のキャラクターによる掛け合い（合成音声） |
| 音声合成 | VOICEVOX（白上虎太郎・四国めたん） |
| 画像生成 | Gemini API（gemini-3.1-flash-image-preview） |
| 動画制作 | Remotion（React ベースの動画フレームワーク） |
| 更新頻度 | 週2本（火曜・金曜 20:00 JST） |

テーマ発見から YouTube 公開まで、ほぼすべての工程を Claude Code が自律的に実行しています。人間が行うのは、Deep Research の実行と、PR ベースのレビュー・承認のみです。

## 制作パイプラインの全体像

```
Phase 1: ネタ収集・テーマ発見    [自動 + ユーザー判断]
    ↓
Phase 2: リサーチ               [手動 + 半自動]
    ↓
Phase 3: 台本制作               [自動 + ユーザー確認]
    ↓
Phase 4: 素材制作（音声・画像）  [自動 + ユーザー確認]
    ↓
Phase 5: 動画制作               [自動 + ユーザー確認]
    ↓
Phase 6: 公開                   [自動 + ユーザー最終判定]
    ↓
Phase 7: 振り返り               [半自動]
```

各工程の自動化レベルは以下の4段階で定義しています。

| レベル | 意味 | 例 |
|--------|------|-----|
| 自動 | Claude Code が単独で実行 | 音声合成・画像生成・レンダリング |
| 半自動 | AI が下書き。人間が確認・修正 | 台本生成・メタデータ作成 |
| 手動 | 人間が操作。AI はツールとして利用 | Deep Research の実行 |
| 承認 | 人間が確認して次に進める判断ポイント | PR の Approve・公開判定 |

:::message
**判断に迷ったときの優先順位**

1. **正確性**: ファクトチェックで不確実な情報があれば止める
2. **品質の一貫性**: キャラクター設定に合わない言動は削除
3. **尺（8〜20分）**: YouTube ミッドロール広告の閾値（8分）を下回らない
4. **自動化の効率**: 品質を落とすくらいなら手動フォールバック
:::

## 対象読者と前提知識

### 対象読者

- Claude Code を実践的なプロジェクトに活用したい方
- AI エージェントを使った複雑なワークフロー構築に興味がある方
- 合成音声・AI 画像生成を動画制作に活用したい方

### 前提知識

| 分野 | 必要レベル |
|------|-----------|
| **Git / GitHub** | ブランチ、PR、Issue の基本操作ができる |
| **Python** | 基本的な文法がわかる（スクリプトの読解程度） |
| **JavaScript / TypeScript / React** | 基本的な文法がわかる（Remotion は React ベース。JSX コンポーネントの読み書きができる程度） |
| **Claude Code** | 基本的なインストール・実行ができる |

:::message alert
**YouTube チャンネルの運営経験は不要です**。YouTube Data API の設定から公開スケジュール管理まで、ゼロからステップバイステップで解説します。
:::
