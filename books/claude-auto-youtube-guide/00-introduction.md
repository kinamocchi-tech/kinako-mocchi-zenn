## この本を手に取ったあなたへ

こんにちは。本書を手に取っていただき、ありがとうございます。

「YouTubeチャンネルを作りたいけれど、動画制作が大変すぎる」
「AIを活用してコンテンツを自動化できると聞いたけど、技術が難しそう…」
「プログラミングは全くわからないけど、自分でも再現できるのかな？」

そんな方のために書いたのが本書です。

## この本でできること

本書では、**Claudeのクラウド環境だけで動くYouTubeチャンネル自動制作システム** の構築方法を、
非エンジニアの方でも再現できるよう丁寧に解説します。

システムが完成したあとの、あなたの主な作業は「監視」だけです。かつては動画1本ごとにあなた自身がディープリサーチを回していましたが、2026年7月時点ではその工程も Claude が自動で実行します。あなたに残るのは、週に一度 Slack のレポートに目を通すことと、必要なら変更差分を確認することだけです。

| 作業 | 担当 | 頻度 |
|------|------|------|
| Slack週次レポートの確認 | **あなた** | 週1回（5分） |
| 変更差分（PR）の確認 | **あなた** | 週1回 |
| テーマのディープリサーチ | Claude が自動実行（research-runner） | 動画1本ごと |
| 台本の生成 | Claude が自動実行 | 毎日 06:00 |
| 音声の合成 | Claude が自動実行 | 毎日 10:00 |
| 画像の生成 | Claude が自動実行 | 毎日 12:00 |
| 動画の編集・書き出し | Claude が自動実行 | 毎日 14:00 |
| YouTubeへのアップロード | Claude が自動実行 | 毎日 14:00〜 |
| SNS配信（Qiita・X・Bluesky・Zenn・note・Instagram Reels） | Claude が自動実行 | 公開後自動 |

:::message
最初の1本だけは、あなた自身の手でディープリサーチを回してみることもできます。仕組みを一度自分の目で追っておくと、自動化に任せたあとも中身を把握しやすくなります（第12章）。
:::

## この本の読み方

本書は **Part 1（基礎知識）→ Part 2（セットアップ）→ Part 3（パイプライン詳解）→ Part 4（自律運用）** の順に
構成されています。

技術用語に不安のある方は Part 1 を丁寧に読んでください。
ある程度理解している方は第8章（セットアップ Step 1）から直接始めても構いません。

## 前提条件

本書を読み進めるにあたって必要なのは以下だけです。

- **PCまたはMacとインターネット接続**（スマートフォンのみでは一部作業が難しい箇所があります）
- **Claude Pro/Max プランへの加入**（Scheduled Tasks と Research 機能の利用に必要）
- **クレジットカード**（各種APIサービスの無料枠/有料プランに使用）
- **プログラミングの知識は不要**

それでは、はじめましょう。
