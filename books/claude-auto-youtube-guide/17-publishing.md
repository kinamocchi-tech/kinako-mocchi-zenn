# Phase 6 公開・SNS配信 ── YouTube APIで全自動アップロード

## このフェーズでやること

Phase 6では、完成動画をYouTubeにアップロードし、スケジュール公開・SNS配信まで自動実行します。

実行タイミング: 動画制作完了後（14:00〜）に自動実行

## YouTube APIによる自動アップロード

Phase 5（動画制作）で `tools/upload_video.py` が動画を YouTube に **限定公開** でアップロードした後、
`youtube_scheduler.py` が以下を自動実行します。

1. **メタデータの設定**: タイトル・説明文・タグ・サムネイル
2. **スケジュール設定**: `publishAt` で自動公開日時を設定
3. **字幕ファイルのアップロード**: 日本語・英語字幕を自動添付

## 動画メタデータの自動生成

YouTube SEOに重要なメタデータは自動で生成されます。

### タイトルの生成ルール

```
基本形: 「{テーマ} ── {サブテーマ}【{キャラクター名}のテック深掘り】」
例: 「Claude Sonnet 4.6 Adaptive Thinking 完全解説 ── AIが自分で考える時代へ【きなこもっちーのテック深掘り】」
```

### 説明文の自動生成

説明文には以下が自動で含まれます:
- 動画の概要（リサーチをもとに生成）
- 目次（タイムスタンプ付き）
- チャンネル登録の案内
- 参考文献・参考リンク一覧

### タグの自動設定

リサーチ内容から関連タグを自動抽出します。
SEOキーワードも考慮してタグを選定します。

## 公開スケジュールの自動管理

**バッファ管理** により、動画は適切な間隔で公開されます。

```
公開済み動画: 週2本を維持
  ↓
バッファ（限定公開で待機中）: 最低2本を常にストック
  ↓
バッファが少なくなったら: 制作を自動加速
```

ユーザーが手動でスケジュールを設定する必要はありません。

## SNS自動配信

動画公開後、`sns-publisher` スキルが以下に自動投稿します。

| プラットフォーム | 投稿内容 |
|----------------|---------|
| **X（Twitter）** | 動画タイトル + 概要 + YouTube URL |
| **Bluesky** | 同上 |
| **Zenn** | 台本をもとにした技術記事 |
| **note** | 動画の深掘り有料記事 |
| **YouTube Shorts** | 動画のハイライト（60秒以内） |

## YouTube Shortsの自動生成

本書のシステムは **YouTube Shorts** も自動生成します。

長尺動画のハイライトを切り出すか、
テック豆知識を短く紹介する形式で、週3〜5本のShortsを自動生成・公開します。

Shortsは登録者獲得に効果的なため、チャンネル成長に貢献します。

:::message
**Shortsの視聴時間とYPP収益化について**
YouTube Shortsの視聴時間は、YPP（YouTube パートナープログラム）Stage 1の
「3,000時間」要件にカウントされません。
長尺動画（8〜20分）の視聴時間が「3,000時間」達成のカギとなります。
:::
