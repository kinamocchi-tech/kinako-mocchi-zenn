# まとめ ── カスタマイズして自分のチャンネルに

## あなたがやることの確認

本書のシステムが完成した後、あなたの定期作業は「監視」だけです。

1. **週1回**: Slackの週次レポートを確認（5分）
2. **週1回**: 気になれば変更差分（PR）に目を通す

これだけです。かつては動画1本ごとに手作業だったディープリサーチも、2026年7月時点では制作の全工程とともに Claude のクラウドが自律的に回します。

## カスタマイズのヒント

本書の「きなこもっちーのテック深掘り」チャンネルの設定をベースに構築しましたが、
あなた独自のチャンネルに合わせてカスタマイズできます。

### キャラクターの変更

`docs/character-sheet.md` と `docs/design-spec.md` を書き換えます。

- 動物キャラクターでなくても構いません
- VOICEVOX以外の音声合成エンジンにも対応可能（設定変更が必要）
- 掛け合いではなく、1キャラクターのナレーション形式にも変更できます

### ジャンルの変更

テック解説チャンネル以外にも応用できます。

- ビジネス解説チャンネル
- 料理レシピチャンネル（ただし動画制作部分は大幅変更が必要）
- 英語学習チャンネル

`CLAUDE.md` と `docs/rules/hourly-routing.md` を書き換えることで、
異なるジャンルにも対応できます。

### 公開頻度の変更

2026年7月時点のデフォルトは、平日毎日 20:00 JST の **公開週5本** です。制作は長尺で週3本ペースを保ち、貯めた在庫のバッファで週5公開を支えています。

- **縮退**: バッファが減ってきたら `tools/youtube_scheduler.py` の `PUBLISH_DAYS` を書き換え、週3本や週2本へ落とします
- **増やす**: 公開ペースを上げるほど在庫バッファに余裕が要り、Max プランでも実行回数には限界があります

## このシステムをさらに発展させるために

本書で構築したシステムは、継続して使うことで成長します。

- **lessons.md が充実する**: ミスが記録・修正されていき、品質が向上する
- **発音辞書が充実する**: 音声の自然さが向上する
- **CLAUDE.mdが育つ**: チャンネル特有のルールが蓄積される

「作って終わり」ではなく、「使うほど育つ」システムです。
